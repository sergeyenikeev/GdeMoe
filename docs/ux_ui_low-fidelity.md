# UX low-fidelity вайрфреймы

1) Quick Add (bottom sheet)
- Поле «Название», чип выбора/создания локации, кнопки «Фото/Видео», переключатель «Сохранить и следующая», кнопка «Сохранить».

2) Batch Add
- Хедер с выбранной локацией и прогрессом «N из M».
- Сетка превью добавленных кадров, кнопки «Снять ещё» и «Готово».

3) Дерево локаций
- Строка поиска, список с отступами по глубине.
- Кнопки «+» и «⋯» справа (добавить/переименовать/переместить/удалить).
- Кнопка «Добавить верхний уровень» внизу.

4) AI Review
- Карточка с превью фото/видео и bbox-оверлеем.
- Текст уверенности и подсказка локации.
- Кнопки «Принять», «Отклонить», «Связать», «Новая вещь», «Исправить локацию».
- Строка лога под карточкой.

5) Карточка вещи
- Карусель медиа, статус, путь локации, кнопка «Переместить», блок заметок.

6) Список вещей
- Строка поиска, фильтры по статусу/тегам, бейдж «Требует проверки».
