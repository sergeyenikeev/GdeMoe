‚Äì –ü—Ä–µ–≤—å—é –≤—ã–±—Ä–∞–Ω–Ω–æ–≥–æ —Ñ–æ—Ç–æ.

–ë–ª–æ–∫ "–û—Å–Ω–æ–≤–Ω–æ–µ":
‚Äì –ù–∞–∑–≤–∞–Ω–∏–µ (–æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ–µ –ø–æ–ª–µ).
‚Äì –ö–∞—Ç–µ–≥–æ—Ä–∏—è (–≤—ã–ø–∞–¥–∞—é—â–∏–π —Å–ø–∏—Å–æ–∫ –∏–ª–∏ —á–∏–ø—ã).
‚Äì –°—Ç–∞—Ç—É—Å (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é, –Ω–∞–ø—Ä–∏–º–µ—Ä, "–ù–æ–≤—ã–π" –∏–ª–∏ "–í –ø–æ—Ä—è–¥–∫–µ").

–ë–ª–æ–∫ "–ú–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ":
‚Äì –°—Ç—Ä–æ–∫–∞ —Å —Ç–µ–∫—É—â–∏–º –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ–º (–Ω–∞–ø—Ä–∏–º–µ—Ä: "–î–æ–º ‚Üí –ö–≤–∞—Ä—Ç–∏—Ä–∞ ‚Üí –ö–æ–º–Ω–∞—Ç–∞ ‚Üí –®–∫–∞—Ñ").
‚Äì –ö–Ω–æ–ø–∫–∞ "–ò–∑–º–µ–Ω–∏—Ç—å" ‚Üí –ø–µ—Ä–µ—Ö–æ–¥ –Ω–∞ —ç–∫—Ä–∞–Ω –≤—ã–±–æ—Ä–∞ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏—è.

–ë–ª–æ–∫ "–î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ":
‚Äì –¢–µ–≥–∏;
‚Äì –î–∞—Ç–∞ –ø–æ–∫—É–ø–∫–∏;
‚Äì –¶–µ–Ω–∞;
‚Äì –ú–∞–≥–∞–∑–∏–Ω;
‚Äì (–ø—Ä–∏ –∂–µ–ª–∞–Ω–∏–∏) —Å–µ—Ä–∏–π–Ω—ã–π –Ω–æ–º–µ—Ä, –º–æ–¥–µ–ª—å, –∑–∞–º–µ—Ç–∫–∞.

–¢—Ä–µ–±–æ–≤–∞–Ω–∏–µ:
–°–¥–µ–ª–∞—Ç—å —Ç–∞–∫, —á—Ç–æ–±—ã –ø—Ä–∏ –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–º –≤–≤–æ–¥–µ (–Ω–∞–∑–≤–∞–Ω–∏–µ + –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ) –∫–∞—Ä—Ç–æ—á–∫–∞ —É–∂–µ –±—ã–ª–∞ –ø—Ä–∏–≥–æ–¥–Ω–∞ –∫ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—é; –æ—Å—Ç–∞–ª—å–Ω—ã–µ –ø–æ–ª—è ‚Äì –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω–æ –∏ –Ω–µ –ø–µ—Ä–µ–≥—Ä—É–∂–∞—é—Ç —ç–∫—Ä–∞–Ω.

4.4. –≠–∫—Ä–∞–Ω –≤—ã–±–æ—Ä–∞ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏—è (–∏–µ—Ä–∞—Ä—Ö–∏—è)

–û—Ç–¥–µ–ª—å–Ω—ã–π —ç–∫—Ä–∞–Ω –∏–ª–∏ –º–æ–¥–∞–ª—å–Ω–æ–µ –æ–∫–Ω–æ:

‚Äì –°–ø–∏—Å–æ–∫ –æ–±—ä–µ–∫—Ç–æ–≤ –≤–µ—Ä—Ö–Ω–µ–≥–æ —É—Ä–æ–≤–Ω—è (–Ω–∞–ø—Ä–∏–º–µ—Ä: "–ö–≤–∞—Ä—Ç–∏—Ä–∞", "–î–∞—á–∞", "–ì–∞—Ä–∞–∂").
‚Äì –ü—Ä–∏ –≤—ã–±–æ—Ä–µ –æ–±—ä–µ–∫—Ç–∞ ‚Äì –ø–µ—Ä–µ—Ö–æ–¥ –Ω–∞ —Å–ª–µ–¥—É—é—â–∏–π —É—Ä–æ–≤–µ–Ω—å (–∫–æ–º–Ω–∞—Ç—ã, –∑–æ–Ω—ã, —à–∫–∞—Ñ—ã –∏ —Ç.–ø.).
‚Äì –ù–∞ –∫–∞–∂–¥–æ–º —É—Ä–æ–≤–Ω–µ:
‚Äì —Å–ø–∏—Å–æ–∫ —ç–ª–µ–º–µ–Ω—Ç–æ–≤;
‚Äì –∫–Ω–æ–ø–∫–∞ "–î–æ–±–∞–≤–∏—Ç—å" (–Ω–∞–ø—Ä–∏–º–µ—Ä: "–î–æ–±–∞–≤–∏—Ç—å –∫–æ–º–Ω–∞—Ç—É", "–î–æ–±–∞–≤–∏—Ç—å —à–∫–∞—Ñ").
‚Äì –í –≤–µ—Ä—Ö–Ω–µ–π —á–∞—Å—Ç–∏ ‚Äì –ø—É—Ç—å (–Ω–∞–ø—Ä–∏–º–µ—Ä: "–î–æ–º ‚Üí –ö–≤–∞—Ä—Ç–∏—Ä–∞ ‚Üí –ö–æ–º–Ω–∞—Ç–∞").

–ù—É–∂–Ω–∞ –ø–æ–Ω—è—Ç–Ω–∞—è –∫–Ω–æ–ø–∫–∞ "–í—ã–±—Ä–∞—Ç—å —ç—Ç–æ –º–µ—Å—Ç–æ" –Ω–∞ –ª—é–±–æ–º —É—Ä–æ–≤–Ω–µ, —á—Ç–æ–±—ã –º–æ–∂–Ω–æ –±—ã–ª–æ –ø—Ä–∏–≤—è–∑–∞—Ç—å –≤–µ—â—å –∫ –Ω—É–∂–Ω–æ–π –≥–ª—É–±–∏–Ω–µ.

4.5. –≠–∫—Ä–∞–Ω "–î–æ–±–∞–≤–∏—Ç—å –ø–æ —Ñ–æ—Ç–æ/–≤–∏–¥–µ–æ"

–î–≤–∞ —Å–æ—Å—Ç–æ—è–Ω–∏—è:

–≠–∫—Ä–∞–Ω –∫–∞–º–µ—Ä—ã:
‚Äì –ö—Ä—É–ø–Ω–∞—è –∫–Ω–æ–ø–∫–∞ —Å—ä–µ–º–∫–∏.
‚Äì –ü–µ—Ä–µ–∫–ª—é—á–∞—Ç–µ–ª—å "–§–æ—Ç–æ" / "–í–∏–¥–µ–æ".
‚Äì –ö–æ—Ä–æ—Ç–∫–∞—è –ø–æ–¥—Å–∫–∞–∑–∫–∞: "–°–Ω–∏–º–∏—Ç–µ –ø–æ–ª–∫—É –∏–ª–∏ —è—â–∏–∫ —Ç–∞–∫, —á—Ç–æ–±—ã –≤–µ—â–∏ –±—ã–ª–∏ —Ö–æ—Ä–æ—à–æ –≤–∏–¥–Ω—ã."

–≠–∫—Ä–∞–Ω –ø–æ–¥—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏—è:
‚Äì –ü—Ä–µ–≤—å—é —Å–¥–µ–ª–∞–Ω–Ω–æ–≥–æ —Ñ–æ—Ç–æ/–≤–∏–¥–µ–æ.
‚Äì –ü–æ–ª–µ –¥–ª—è –≤—ã–±–æ—Ä–∞ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏—è (–∫—É–¥–∞ –æ—Ç–Ω–æ—Å–∏—Ç—Å—è —ç—Ç–æ –º–µ–¥–∏–∞).
‚Äì –ö–Ω–æ–ø–∫–∞ "–û—Ç–ø—Ä–∞–≤–∏—Ç—å –Ω–∞ –∞–Ω–∞–ª–∏–∑".

–î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ ‚Äì —Å–ø–∏—Å–æ–∫ –ø–æ—Å–ª–µ–¥–Ω–∏—Ö –º–µ–¥–∏–∞ –∏ –∏—Ö —Å—Ç–∞—Ç—É—Å—ã:
‚Äì "–ê–Ω–∞–ª–∏–∑–∏—Ä—É–µ—Ç—Å—è";
‚Äì "–ì–æ—Ç–æ–≤–æ ‚Äì –Ω–∞–π–¥–µ–Ω–æ N –≤–µ—â–µ–π";
‚Äì "–û—à–∏–±–∫–∞ –∞–Ω–∞–ª–∏–∑–∞" (—Å –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å—é –ø–æ–≤—Ç–æ—Ä–∏—Ç—å).

4.6. –≠–∫—Ä–∞–Ω "–ù–∞–¥–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å" (—Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã –ò–ò, —Ç—Ä–µ–±—É—é—â–∏–µ –ø—Ä–æ–≤–µ—Ä–∫–∏)

–≠—Ç–æ –æ–¥–∏–Ω –∏–∑ –∫–ª—é—á–µ–≤—ã—Ö —ç–∫—Ä–∞–Ω–æ–≤.

–û–ø–∏—Å–∞–Ω–∏–µ:
‚Äì –°–ø–∏—Å–æ–∫ –∑–∞–ø–∏—Å–µ–π, –∫–∞–∂–¥–∞—è –∑–∞–ø–∏—Å—å ‚Äì –æ–¥–∏–Ω "–ø—Ä–µ–¥–º–µ—Ç-–∫–∞–Ω–¥–∏–¥–∞—Ç" –Ω–∞–π–¥–µ–Ω–Ω—ã–π –ò–ò.
‚Äì –î–ª—è –∫–∞–∂–¥–æ–π –∑–∞–ø–∏—Å–∏:
‚Äì –º–∏–Ω–∏-–ø—Ä–µ–≤—å—é –æ–±—ä–µ–∫—Ç–∞ (–∫—Ä–æ–ø –∏–ª–∏ –∏–∫–æ–Ω–∫–∞);
‚Äì –ø—Ä–µ–¥–ª–∞–≥–∞–µ–º–æ–µ –Ω–∞–∑–≤–∞–Ω–∏–µ –∏–ª–∏ –ø–æ–¥–ø–∏—Å—å "–ù–æ–≤–∞—è –≤–µ—â—å";
‚Äì –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—è: "–ü–æ—Ö–æ–∂ –Ω–∞: [–Ω–∞–∑–≤–∞–Ω–∏–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–µ–π –∫–∞—Ä—Ç–æ—á–∫–∏]" –∏–ª–∏ "–ù–µ –Ω–∞—à–ª–∏ —Å–æ–≤–ø–∞–¥–µ–Ω–∏–π";
‚Äì —Å—Ç–∞—Ç—É—Å: "–ù–∞–¥–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å".

–ü—Ä–∏ –≤—ã–±–æ—Ä–µ –∑–∞–ø–∏—Å–∏ ‚Äì –¥–µ—Ç–∞–ª—å–Ω—ã–π —ç–∫—Ä–∞–Ω –ø—Ä–æ–≤–µ—Ä–∫–∏:

‚Äì –ö—Ä—É–ø–Ω–æ–µ —Ñ–æ—Ç–æ –æ–±—ä–µ–∫—Ç–∞.
‚Äì –ë–ª–æ–∫ "–ß—Ç–æ —ç—Ç–æ –∑–∞ –≤–µ—â—å?":
‚Äì –≤–∞—Ä–∏–∞–Ω—Ç 1: —ç—Ç–æ —Å—É—â–µ—Å—Ç–≤—É—é—â–∞—è –≤–µ—â—å ‚Üí —Å–ø–∏—Å–æ–∫ –ø—Ä–µ–¥–ª–∞–≥–∞–µ–º—ã—Ö –∫–∞—Ä—Ç–æ—á–µ–∫ (—Å —Ä–µ–π—Ç–∏–Ω–≥–æ–º —É–≤–µ—Ä–µ–Ω–Ω–æ—Å—Ç–∏);
‚Äì –≤–∞—Ä–∏–∞–Ω—Ç 2: —ç—Ç–æ –Ω–æ–≤–∞—è –≤–µ—â—å ‚Üí –∫–Ω–æ–ø–∫–∞ "–°–æ–∑–¥–∞—Ç—å –Ω–æ–≤—É—é –∫–∞—Ä—Ç–æ—á–∫—É" —Å –ø—Ä–µ–¥–∑–∞–ø–æ–ª–Ω–µ–Ω–Ω—ã–º –Ω–∞–∑–≤–∞–Ω–∏–µ–º/—Ñ–æ—Ç–æ.
‚Äì –ë–ª–æ–∫ "–ú–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ":
‚Äì —Ç–µ–∫—É—â–∞—è –ø—Ä–∏–≤—è–∑–∫–∞ (–ª–∏–±–æ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–æ –ò–ò, –ª–∏–±–æ –≤—ã–±—Ä–∞–Ω–Ω–æ–µ —Ä–∞–Ω–µ–µ –º–µ—Å—Ç–æ);
‚Äì –∫–Ω–æ–ø–∫–∞ "–ò–∑–º–µ–Ω–∏—Ç—å" ‚Üí –ø–µ—Ä–µ—Ö–æ–¥ –Ω–∞ —ç–∫—Ä–∞–Ω –≤—ã–±–æ—Ä–∞ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏—è.
‚Äì –ë–æ–ª—å—à–∏–µ –∫–Ω–æ–ø–∫–∏ –≤–Ω–∏–∑—É:
‚Äì "–ü–æ–¥—Ç–≤–µ—Ä–¥–∏—Ç—å" (—Å–æ—Ö—Ä–∞–Ω–∏—Ç—å —Ç–µ–∫—É—â–∏–π –≤—ã–±–æ—Ä);
‚Äì "–û—Ç–ª–æ–∂–∏—Ç—å" (–æ—Å—Ç–∞–≤–∏—Ç—å –≤ —Å–ø–∏—Å–∫–µ "–ù–∞–¥–æ –ø—Ä–æ–≤–µ—Ä–∏—Ç—å" –Ω–∞ –ø–æ—Ç–æ–º).

–í–∞–∂–Ω–æ –ø—Ä–µ–¥—É—Å–º–æ—Ç—Ä–µ—Ç—å —Ä–µ–∂–∏–º –±—ã—Å—Ç—Ä–æ–≥–æ –ø—Ä–æ–ª–∏—Å—Ç—ã–≤–∞–Ω–∏—è:
‚Äì –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ –æ–¥–Ω–æ–≥–æ –∫–∞–Ω–¥–∏–¥–∞—Ç–∞ –Ω–∞ —ç–∫—Ä–∞–Ω–µ;
‚Äì —Å–≤–∞–π–ø –≤–ø—Ä–∞–≤–æ ‚Äì "–ü–æ–¥—Ç–≤–µ—Ä–¥–∏—Ç—å", –≤–ª–µ–≤–æ ‚Äì "–û—Ç–ª–æ–∂–∏—Ç—å";
‚Äì –∏–Ω–¥–∏–∫–∞—Ç–æ—Ä "–û—Å—Ç–∞–ª–æ—Å—å N".

4.7. –≠–∫—Ä–∞–Ω –∫–∞—Ä—Ç–æ—á–∫–∏ –≤–µ—â–∏ (–ø—Ä–æ—Å–º–æ—Ç—Ä –∏ —Ä–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ)

–°—Ç—Ä—É–∫—Ç—É—Ä–∞:

–í–µ—Ä—Ö–Ω–∏–π –±–ª–æ–∫:
‚Äì –≥–ª–∞–≤–Ω–æ–µ —Ñ–æ—Ç–æ;
‚Äì –Ω–∞–∑–≤–∞–Ω–∏–µ;
‚Äì —Å—Ç–∞—Ç—É—Å (—Ü–≤–µ—Ç–æ–≤–æ–π –∏–Ω–¥–∏–∫–∞—Ç–æ—Ä);
‚Äì –∫—Ä–∞—Ç–∫–æ–µ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ –æ–¥–Ω–æ–π —Å—Ç—Ä–æ–∫–æ–π.

–û—Å–Ω–æ–≤–Ω—ã–µ –±–ª–æ–∫–∏ –Ω–∏–∂–µ:
‚Äì "–ú–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ": –ø–æ–ª–Ω–æ–µ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ –ø—É—Ç–∏, –∫–Ω–æ–ø–∫–∞ "–ò–∑–º–µ–Ω–∏—Ç—å".
‚Äì "–ü–∞—Ä–∞–º–µ—Ç—Ä—ã": –º–æ–¥–µ–ª—å, —Å–µ—Ä–∏–π–Ω—ã–π –Ω–æ–º–µ—Ä, –¥–∞—Ç–∞ –ø–æ–∫—É–ø–∫–∏, —Ü–µ–Ω–∞, –º–∞–≥–∞–∑–∏–Ω –∏ –¥—Ä.
‚Äì "–ú–µ–¥–∏–∞": —Å–µ—Ç–∫–∞ –º–∏–Ω–∏–∞—Ç—é—Ä —Ñ–æ—Ç–æ –∏ –≤–∏–¥–µ–æ; –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –æ—Ç–∫—Ä—ã—Ç—å –º–µ–¥–∏–∞.
‚Äì "–ó–∞–º–µ—Ç–∫–∏": —Å–ø–∏—Å–æ–∫ —Ç–µ–∫—Å—Ç–æ–≤—ã—Ö –∑–∞–º–µ—Ç–æ–∫, –∫–Ω–æ–ø–∫–∞ "–î–æ–±–∞–≤–∏—Ç—å –∑–∞–º–µ—Ç–∫—É".
‚Äì "–ò—Å—Ç–æ—Ä–∏—è": –ª–µ–Ω—Ç–∞ –∏–∑–º–µ–Ω–µ–Ω–∏–π (—Å–æ–∑–¥–∞–Ω–æ, –ø–µ—Ä–µ–º–µ—â–µ–Ω–æ, —Å–º–µ–Ω–∞ —Å—Ç–∞—Ç—É—Å–∞ –∏ —Ç.–ø.).

–ö–Ω–æ–ø–∫–∏ –¥–µ–π—Å—Ç–≤–∏–π:
‚Äì "–†–µ–¥–∞–∫—Ç–∏—Ä–æ–≤–∞—Ç—å";
‚Äì "–î–æ–±–∞–≤–∏—Ç—å —Ñ–æ—Ç–æ";
‚Äì "–ü–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å";
‚Äì "–ò–∑–º–µ–Ω–∏—Ç—å —Å—Ç–∞—Ç—É—Å".

4.8. –≠–∫—Ä–∞–Ω –ø—Ä–æ—Å–º–æ—Ç—Ä–∞ —Ñ–æ—Ç–æ/–≤–∏–¥–µ–æ –∏ —Å–≤—è–∑–µ–π

–ü—Ä–∏ –æ—Ç–∫—Ä—ã—Ç–∏–∏ —Ñ–æ—Ç–æ –∏–ª–∏ –≤–∏–¥–µ–æ:

‚Äì –ü–æ–ª–Ω–æ—ç–∫—Ä–∞–Ω–Ω–æ–µ –æ—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ –º–µ–¥–∏–∞.
‚Äì –ü–∞–Ω–µ–ª—å —Å –æ—Å–Ω–æ–≤–Ω–æ–π –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–µ–π:
‚Äì —Å–ø–∏—Å–æ–∫ —Å–≤—è–∑–∞–Ω–Ω—ã—Ö –∫–∞—Ä—Ç–æ—á–µ–∫ (–ø–æ —Ç–∞–ø—É ‚Äì –ø–µ—Ä–µ—Ö–æ–¥ –∫ –∫–∞—Ä—Ç–æ—á–∫–µ);
‚Äì —Ç–µ–∫—É—â–µ–µ –º–µ—Å—Ç–æ–ø–æ–ª–æ–∂–µ–Ω–∏–µ (–µ—Å–ª–∏ –º–µ–¥–∏–∞ –ø—Ä–∏–≤—è–∑–∞–Ω–æ –∫ –º–µ—Å—Ç—É);
‚Äì –∫–Ω–æ–ø–∫–∞ "–ò–∑–º–µ–Ω–∏—Ç—å —Å–≤—è–∑–∏" (–¥–æ–±–∞–≤–∏—Ç—å –∏–ª–∏ —É–¥–∞–ª–∏—Ç—å –ø—Ä–∏–≤—è–∑–∫–∏ –∫ –∫–∞—Ä—Ç–æ—á–∫–∞–º –∏/–∏–ª–∏ –º–µ—Å—Ç–∞–º).

4.9. –≠–∫—Ä–∞–Ω –ø–æ–∏—Å–∫–∞

–°–æ—Å—Ç–∞–≤:

‚Äì –°—Ç—Ä–æ–∫–∞ –ø–æ–∏—Å–∫–∞ (–ø–æ –Ω–∞–∑–≤–∞–Ω–∏—é –∏ —Ç–µ–∫—Å—Ç—É).
‚Äì –ù–∏–∂–µ ‚Äì –Ω–∞–±–æ—Ä —Ñ–∏–ª—å—Ç—Ä–æ–≤ –≤ –≤–∏–¥–µ "—á–∏–ø–æ–≤":
‚Äì —Ñ–∏–ª—å—Ç—Ä –ø–æ —Å—Ç–∞—Ç—É—Å—É;
‚Äì —Ñ–∏–ª—å—Ç—Ä –ø–æ –¥–æ–º—É/–∫–≤–∞—Ä—Ç–∏—Ä–µ;
