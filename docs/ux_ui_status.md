# Статус UX/UI

Актуально на 2025-12-11
- Навигация: bottom bar (Вещи, Локации, Добавить, AI, Настройки) — реализовано.
- Quick Add: одно поле, выбор/поиск локации, медиа, «сохранить и следующая» — реализовано (нужна полировка UI).
- Batch Add: концепт готов, требуется реализация экрана.
- Локации: дерево с действиями добавить/переименовать/переместить/удалить — базовый экран есть, нужны операции.
- AI Review: инбокс с превью медиа, accept/reject, кандидаты — реализовано, нужен полноценный media viewer и привязка к location picker.
- Медиа: загрузка/привязка есть, полноэкранный просмотр и оффлайн-кеш — в работе.
- Документация: задачи/концепт/low-fi восстановлены после проблем с кодировкой.
