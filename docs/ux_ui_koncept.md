# UX-концепт "ГдеМоё"

Цели: уложиться в 10–20 секунд на добавление вещи; простая навигация, понятное положение вещи.

Основные принципы
- Навигация: bottom bar — Вещи, Локации, Добавить (центр), AI, Профиль/Настройки.
- Экран «Вещи»: список с поиском/фильтрами, карточка с медиа-каруселью и кнопкой «Переместить».
- «Локации»: дерево дом → квартира → комната → шкаф → ящик → коробка, быстрые действия Добавить/Переименовать/Переместить/Удалить.
- «Добавить»: быстрый ввод (название + локация + медиа, чекбокс «Сохранить и следующая») и батч-добавление (серия фото/видео из одной коробки).
- «AI»: инбокс предложений, уверенность, подсказка локации, кнопки Принять/Отклонить/Связать/Создать, лог решений.
- Медиа: карусель превью, полноэкранный просмотр, загрузка в фоне, авто-отправка в AI.

Расширение
- Поддержка тегов, фильтров, сценариев переезда, совместного доступа — закладываются в роуты и модель.
- Сбор статистики действий для улучшения UX.

Тональность интерфейса
- Чёткая иерархия, крупные акценты, минимум лишних полей, понятные кнопки действия.
